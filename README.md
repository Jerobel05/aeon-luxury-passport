# AEON — Passeport Numérique de Luxe

  > Projet académique Web3

  Application web qui authentifie les montres de luxe par **IA visuelle** et crée un **certificat NFT** sur la blockchain.

  ## Technologies
  - React + Vite + TypeScript
  - wagmi v2 + viem v2 (Ethereum)
  - ERC-721 (NFT) + ERC-4337 (Account Abstraction)
  - IPFS (stockage décentralisé)
  - Framer Motion + Tailwind CSS

  ## Pages
  1. **Accueil** — Landing page luxe
  2. **Scanner** — Scan IA de la montre
  3. **Passeport** — Certificat NFT blockchain
  4. **Revente** — Marché secondaire gasless
  5. **Architecture** — Diagrammes UML

  ## Lancer le projet
  ```bash
  pnpm install
  pnpm --filter @workspace/invisible-luxury run dev
  ```
